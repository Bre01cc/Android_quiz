## Sobre
Projeto desenvolvido com o objetivo de colocar em prática conceitos abordados em aulas anteriores, como arquitetura MVVM, navegação entre telas, gerenciamento de estado e composição de componentes.

A partir disso, foi proposto o desenvolvimento de um quiz de conhecimentos gerais, integrando na prática os conceitos estudados ao longo das aulas.

Com base nisso, foi proposto o desenvolvimento de um quiz de conhecimentos gerais que contemple e integre os conteúdos estudados anteriormente.

Disciplina ministrada pelo professor [João Henrique](https://www.linkedin.com/in/joao-henrique-parizoti-meyer/?locale=pt_BR).

## Descrição 
A aplicação consiste em um quiz de três perguntas de conhecimentos gerais que, ao fim
exibe a quantidade de acertos com base nas alternativas selecionadas pelo usuário.
